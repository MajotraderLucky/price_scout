## Проблемы существующих решений

### Для B2C сегмента (покупатели)

1. **Фрагментированность** - нужно использовать разные инструменты для разных маркетплейсов
2. **Отсутствие кросс-платформенного сравнения** - нельзя сравнить цену на один товар на WB vs Ozon
3. **Низкая частота обновлений** - у бесплатных сервисов проверка раз в 3-6 часов
4. **Нет истории цен в удобном виде** - сложно понять, когда товар стоил дешевле всего
5. **Ограниченные способы уведомлений** - обычно только email или Telegram
6. **Нет умных рекомендаций** - не предлагают похожие товары дешевле

### Для B2B сегмента (продавцы)

1. **Высокая стоимость** - от 3000 руб/мес за базовый функционал
2. **Избыточность** - продают комплексную аналитику, когда нужен только мониторинг цен
3. **Сложность интеграции** - не все предоставляют API для автоматизации

## Конкурентные преимущества нашего сервиса

### Основная идея

**Target Products** - сервис отслеживания цен на **технику и электронику** с кросс-платформенным сравнением.

### Целевая ниша: Техника и электроника

| Почему техника                     | Преимущество                                 |
|------------------------------------|----------------------------------------------|
| Высокий средний чек                | 10-100k руб - люди готовы ждать скидку       |
| Чёткие артикулы                    | Samsung S24, Dyson V15, Braun 3010s          |
| Долгий цикл покупки                | Планируют месяцами - есть смысл отслеживать  |
| Сезонные скидки                    | Чёрная пятница, 11.11, Новый год             |
| Одинаковый товар на всех площадках | Легко сравнивать WB vs Ozon vs ЯМ            |

### Уникальные преимущества (USP)

| Преимущество                      | Описание                                        |
|-----------------------------------|-------------------------------------------------|
| [+] Кросс-платформенное сравнение | Один товар - цены на WB, Ozon, Яндекс Маркет    |
| [+] Фокус на технике              | Оптимизировано для электроники с артикулами     |
| [+] Уведомление о минимуме        | Алерт когда цена ниже всего на любой площадке   |
| [+] История цен                   | Графики за месяцы - видно когда было дешевле    |
| [+] Telegram-first                | Удобный бот, не нужно ставить приложение        |
| [+] Self-hosted (позже)           | Можно развернуть на своем сервере               |

### Модель монетизации

| Тариф       | Цена            | Лимиты                           | Функционал                    |
