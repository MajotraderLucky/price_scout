# Price Scout - Документация проекта

## Обзор

**Price Scout** ("Разведчик цен") - сервис мониторинга и сравнения цен на товары с различных маркетплейсов и интернет-магазинов.

## Быстрый старт

```bash
# Клонирование
git clone <repo-url>
cd price_scout

# Сборка
cargo build --release

# Запуск
./target/release/price_scout
```

## Ключевые особенности

- Telegram-бот для взаимодействия с пользователями
- Парсинг агрегаторов цен (E-katalog)
- Сравнение цен на один товар из разных магазинов
- Уведомления о снижении цены
- История изменения цен

## Контакты

Проект разрабатывается в рамках личного использования.
